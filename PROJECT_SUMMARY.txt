⚙️ SETUP REQUIREMENTS
════════════════════════════════════════════════════════════════════════════════

1. OpenAI API Key (Required for Quiz Generation)
   • Get it from: https://platform.openai.com/api-keys
   • Add to .env file: OPENAI_API_KEY=sk-your-key-here
   • Cost: ~$0.002 per quiz (very cheap!)

2. Python 3.8+ (Required)
   • Check: python3 --version
   • Install if needed

3. Internet Connection (Required for first-time setup)
   • Downloads models & dependencies
   • ~500MB total download

🎨 KEY TECHNOLOGIES
════════════════════════════════════════════════════════════════════════════════

Backend:
  • Python 3.8+
  • LangChain (RAG framework)
  • OpenAI GPT-3.5-turbo (Quiz generation)
  • SentenceTransformers (Embeddings)

Storage:
  • FAISS (Vector database)
  • JSON (Quiz results)
  • Pickle (Model persistence)

Frontend:
  • Streamlit (Web interface)
  • Plotly (Interactive charts)
  • Custom CSS styling

Document Processing:
  • PyPDF2 (PDF extraction)
  • pdfplumber (Enhanced PDF parsing)
  • python-docx (DOCX handling)
